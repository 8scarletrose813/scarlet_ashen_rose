Global Chat Overhaul

  • Rich Media Rendering: Implemented auto-expansion for direct links (jpeg, jpg, gif, png, webp).
  • Dynamic Identity: Added a custom color-hashing function. Your chat color is now uniquely generated from your username (DrBopoH vs Dr.BopoH).
  • UI Refactoring: Complete polish of chat component layout for more intuitive navigation.
  • Experimental Tech: Early-stage integration of Escape Codes (active testing phase).

System & Pipeline Fixes

  • Mobile Responsiveness: Fixed a critical breaking bug where the sidebar killed mobile adaptation. Sidebar toggle is now fully functional on web-mobile.
  • Database Logic Fix: Corrected a high-priority bug in the message pipeline. The system now correctly pulls the Last 100 messages.
